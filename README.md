## 🛡️ Defensive Coding Checklist (시험 전 필독)

1. **시간 복잡도 역산**: $N$의 크기를 보고 알고리즘을 선택했는가?
   - $N \le 100,000 \rightarrow O(N \log N)$
   - $N \le 1,000,000 \rightarrow O(N)$
2. **엣지 케이스**: 입력이 비어있거나(`[]`), 데이터가 1개인 경우 처리했는가?
3. **오버플로우**: 정답이 21억을 넘을 경우 `Long` 타입을 고려했는가?
4. **무한 루프 방지**: `while`문의 종료 조건이 확실한가?
